문제: n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

입력: 첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

출력: 1부터 n까지 합을 출력한다.

#include <iostream>
int main()
{
    int n;
    int sum = 0;
    std::cin >> n;

    for(int i = 1; i <= n; ++i){
        sum += i;
    }
    std::cout << sum << std::endl;
    return 0;
}
여기서 sum을 미리 초기화하지 않을 경우 오류 발생.
