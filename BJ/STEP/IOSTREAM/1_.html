(A+B)%C는 ((A%C)+(B%C))%C와 같을까?
(A*B)%C는 ((A%C)*(B%C))%C와 같을까?
세 수 A, B, C가 주어졌을 때, 위의 네 가지 값을 구하는 프로그램을 작성하시오.

입력: 첫째 줄에 A, B, C가 순서대로 주어진다.

출력: 첫째 줄에 (A+B)%C, 둘째 줄에 ((A%C) + (B%C))%C, 셋째 줄에 (A*B)%C, 넷째 줄에 ((A%C) * (B%C))%C를 출력한다.

#include <iostream>
int main()
{
    int a, b, c;
    std::cin >> a >> b >> c;
    std::cout << (a+b)%c << "\n"
              << ((a%c)+(b%c))%c << "\n"
              << (a*b)%c << "\n"
              << ((a%c)*(b%c))%c << std::endl;
}
