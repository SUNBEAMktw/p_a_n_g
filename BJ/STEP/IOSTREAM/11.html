2558: 곱셈

←

(세 자리 수) * (세 자리 수)는 다음과 같은 과정을 통해 이루어진다.

(1)과 (2)위치에 들어갈 세 자리 자연수가 주어질 때, (3), (4), (5), (6) 위치에 들어갈 값을 구하는 프로그램을 작성하시오.

입력: 첫째 줄에 (1)의 위치에 들어갈 세 자리 자연수가, 둘째 줄에 (2)의 위치에 들어갈 세 자리 자연수가 주어진다.

출력: 첫째 줄부터 넷째 줄까지 차례대로 (3), (4), (5), (6)에 들어갈 값을 출력한다. 

#include <iostream>
int main()
{
    int a, b;
    std::cin >> a >> b;

    int ini = (b%100);
    int fir = ini % 10;
    int sec = (ini - fir)/10;
    int thr = b/100;

    int c = a * fir;
    int d = a * sec;
    int e = a * thr;
    int f = c + d * 10 + e * 100;

    std::cout << c << "\n"
              << d << "\n"
              << e << "\n"
              << f << std::endl;
}
