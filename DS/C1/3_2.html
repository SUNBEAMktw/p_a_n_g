2. Set ADT를 정의하라. 표준 수학 정의를 사용하고 다음 연산을 포함시켜라: Create, Insert, Remove, IsIn, Union, Intersection, Difference.

ADT Set is
      객체: 집합은 객체의 모임.

        Create()                    ::=   집합을 생성하여 반환한다.
        Insert(S, x)                ::=   원소 x를 집합 S에 저장한다.
        Remove(S, x)                ::=   원소 x를 집합 S에서 삭제한다.
        Is_In(S, item)              ::=   집합 S에 x가 있는지 검사한다.
        Union(S1, S2)               ::=   S1과 S2의 합집합을 구한다.
        Intersection(S1, S2)        ::=   S1과 S2의 교집합을 구한다.
        Difference(S1, S2)          ::=   S1과 S2의 차집합을 구한다.
end Set
