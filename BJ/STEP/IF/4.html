문제: 점의 좌표를 입력받아 그 점이 어느 사분면에 속하는지 알아내는 프로그램을 작성하시오. 단, x좌표와 y좌표는 모두 양수나 음수라고 가정한다.

첫 줄에는 정수 x가 주어진다. (−1000 ≤ x ≤ 1000; x ≠ 0) 다음 줄에는 정수 y가 주어진다. (−1000 ≤ y ≤ 1000; y ≠ 0)

점 (x, y)의 사분면 번호(1, 2, 3, 4 중 하나)를 출력한다.p> 

#include 
int main()
{
    int x, y;
    std::cin >> x >> y;

    if ((x>0)&&(y>0))
        std::cout << 1 << std::endl;
    else if ((x<0)&&(y>0))
        std::cout << 2 << std::endl;
    else if ((x<0)&&(y<0))
        std::cout << 3 << std::endl;
    else
        std::cout << 4 << std::endl;
    return 0;
}
