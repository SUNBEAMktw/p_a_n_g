10869: 사칙연산

←

두 자연수 A와 B가 주어진다. 이때, A+B, A-B, A*B, A/B, A%B를 출력하는 프로그램을 작성하시오.

입력: 첫째 줄에 A와 B가 주어진다.(0 < A, B < 10)

출력: 첫째 줄에 A+B, 둘째 줄에 A-B, 셋째 줄에 A*B, 넷째 줄에 A/B, 다섯째 줄에 A%B를 출력한다.

#include <iostream>
int main()
{
    int a, b;
    std::cin >> a >> b;
    std::cout << a + b << "\n" <<
                 a - b << "\n" <<
                 a * b << "\n" <<
                 a / b << "\n" <<
                 a % b << std::endl;
}
